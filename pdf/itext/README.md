# IText PDF Examples

## Create project using maven
```
mvn archetype:generate -DgroupId=com.itext -DartifactId=itext -Dversion=1.0 -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
```

## Add gradle
```
gradle init --type pom
```
