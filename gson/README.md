# Google GSON Library

## Requirement
* Using **gson** library

## Create project using maven
```
mvn archetype:generate -DgroupId=gson -DartifactId=gson -Dversion=1.0 -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
```

## Add gradle
```
gradle init --type pom
```
